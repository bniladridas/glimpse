# GLIMPSE

GLIMPSE is a simple place to make logos and clean up images. It is here to help when you need a quick tool for your brand.

If you need a logo, you can pick an icon and add your text. If you have a photo with a background you do not need, the tool can help you remove it. It tries to stay quiet and out of your way while you work.

The tool works on your computer and your phone. You can also choose between light, dark, or grey looks to make it easy on your eyes. We hope these small tools help you with your tasks.

### Using it on a Mac
If you want to use this like a Mac app, you can open it in Safari and choose "Add to Dock." This puts it right in your application list and gives it a clean window of its own.

### Hosting or Sharing
To share this on Vercel, you can simply connect your code. The tool is already prepared with a config file to handle everything. For GitHub Pages, it works as a static site. To push your code to GitHub, you can use the Share menu in the top right corner of this screen.
